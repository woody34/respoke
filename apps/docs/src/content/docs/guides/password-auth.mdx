Rescope supports the full password authentication lifecycle: signup, signin, update, reset, and replace.

## Sign Up

```bash
curl -X POST http://localhost:4500/v1/auth/password/signup \
  -H 'Content-Type: application/json' \
  -d '{
    "loginId": "alice@example.com",
    "password": "my-secure-password",
    "user": { "name": "Alice" }
  }'
# → {"sessionJwt": "eyJ...", "refreshJwt": "eyJ..."}
```

## Sign In

```bash
curl -X POST http://localhost:4500/v1/auth/password/signin \
  -H 'Content-Type: application/json' \
  -d '{"loginId": "alice@example.com", "password": "my-secure-password"}'
```

## Password Reset

<Steps>

1. **Request a reset**

   ```bash
   curl -X POST http://localhost:4500/v1/auth/password/reset \
     -H 'Content-Type: application/json' \
     -d '{"loginId": "alice@example.com", "URI": "http://localhost:3000/reset"}'
   ```

2. **Retrieve the reset token** (same as magic link)

   ```bash
   curl http://localhost:4500/emulator/otp/alice@example.com
   # → {"code": "eyJ..."}
   ```

3. **Set the new password**

   ```bash
   curl -X POST http://localhost:4500/v1/auth/password/replace \
     -H 'Content-Type: application/json' \
     -d '{"loginId": "alice@example.com", "oldPassword": "old", "newPassword": "new-password"}'
   ```

</Steps>

## SDK example

```typescript
const client = DescopeClient({ projectId: 'Ptest', baseURL: 'http://localhost:4500' });

// Sign up
await client.password.signUp('alice@example.com', 'password123', { name: 'Alice' });

// Sign in
const { data } = await client.password.signIn('alice@example.com', 'password123');
console.log('Token:', data.sessionJwt);
```

## Supported endpoints

| Endpoint | Description |
|---|---|
| `POST /v1/auth/password/signup` | Create account with password |
| `POST /v1/auth/password/signin` | Authenticate with password |
| `POST /v1/auth/password/reset` | Request a password reset link |
| `POST /v1/auth/password/replace` | Replace password (old → new) |
| `POST /v1/auth/password/update` | Update password (authenticated) |
| `GET /v1/auth/password/policy` | Get password policy |
